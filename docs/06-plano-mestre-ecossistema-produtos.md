# Plano Mestre — Ecossistema de Produtos Gisa Ferreira

## Contexto

Gisa Ferreira é estrategista de Posicionamento Pessoal Estratégico para mulheres especialistas. Ela domina branding, imagem, comportamento e negócios — mas **não tem sistemas**. Rafael (Growai) vai criar toda a infraestrutura: esteira de produtos, automações, agentes IA e snapshot vendável, usando a plataforma Growai + OpenClaw/Nora + AIOX Core.

**Nicho oficial**: Mulheres especialistas, consultoras e empreendedoras que vendem conhecimento e precisam transformar expertise em autoridade percebida.

## 1. ESTEIRA DE PRODUTOS (6 camadas)

### 1.1 Isca Digital — "Diagnóstico de Posicionamento" (R$0)
- **Formato**: Quiz interativo (Survey com Scoring + Logic Branching na Growai)
- **12 perguntas** sobre imagem, marca, comportamento, presença digital
- **Resultado em 4 tiers**: Invisível (0-25) → Confusa (26-50) → Percebida (51-75) → Autoridade (76-100)
- **Entrega**: Página de resultado com score visual + 3 dicas por tier + CTA tripwire
- **Automação**: Survey Submitted → Create Contact → Tag tier → Email resultado → Start Nurture

### 1.2 Tripwire — "O Mapa da Autoridade" (R$67)
- **Formato**: Mini-curso 7 lições (15-20min cada) na Membership Area
- **Módulos**: O Erro Fatal → Expertise Única → Imagem de Autoridade → Mensagem em Uma Frase → LinkedIn como Palco → Comportamento de Autoridade → Plano de 30 Dias
- **Drip**: 1 lição/dia (7 dias)
- **Order bump**: "Kit de Templates de Bio" — R$27
- **Funnel**: Landing (VSL 5min) → Order Form → Upsell (core) → Thank You + acesso
- **Automação**: Payment → Tag "cliente-tripwire" → Grant Access → Welcome Email → Day 5: oferta MAP

### 1.3 Core Offer — "Método Autoridade Percebida" / MAP (R$1.997 ou 12x R$197)
- **Formato**: Programa online 8 semanas (vídeo-aulas + exercícios + calls quinzenais)
- **Estrutura**:
  - Semana 1-2: **Pilar Imagem** — Diagnóstico visual, guarda-roupa estratégico
  - Semana 3-4: **Pilar Marca** — Posicionamento, mensagem, bio, identidade visual
  - Semana 5-6: **Pilar Comportamento** — Comunicação, apresentações, networking
  - Semana 7-8: **Pilar Negócio** — Precificação, esteira, LinkedIn, pipeline
- **Calls**: 4 calls de 60min (Group Calendar, quinzenal)
- **Comunidade**: Acesso durante programa + 30 dias após
- **Certificado**: Badge "Autoridade Percebida"
- **Funnel**: Application Survey (scoring) → Calendar booking → Strategy call → Follow-up → Matrícula
- **Pipeline**: Lead → Aplicou → Call Agendada → Call Feita → Proposta → Matriculada → Concluiu

### 1.4 Premium — "Imersão Posicionamento 360°" / IP360 (R$8.997 ou 6x R$1.597)
- **Formato**: Mentoria individual 3 meses (8 sessões 1:1 de 60min + WhatsApp)
- **Sessões**: Diagnóstico completo → Plano Estratégico → Identidade Visual + Guarda-roupa → Mensagem + Pitch → Estratégia Conteúdo → Precificação + Esteira → Media Training → Review final + Plano 90 dias
- **Entregáveis**: Mapa de Posicionamento (PDF), Guia de Estilo, Banco de 30 posts, Plano 90 dias
- **Funnel**: Application-only → Call diagnóstico → Proposta → Pagamento
- **Inclui**: Agentes IA de Diagnóstico + Gerador de Conteúdo como diferencial premium

### 1.5 Recorrência — "Clube Autoridade" (R$97/mês ou R$970/ano)
- 1 masterclass nova/mês (tema rotativo entre 4 pilares)
- 1 call Q&A ao vivo/mês
- Templates mensais (posts, bios, emails, apresentações)
- Acesso à comunidade exclusiva
- Descontos em produtos premium
- **Automação churn**: Past Due → SMS + Email → 3 dias → Task para Gisa ligar
- **Milestones**: 3 meses → Badge, 6 meses → 20% off mentoria, 12 meses → Call 1:1

### 1.6 Comunidade — "Círculo de Autoridade" (incluída para clientes pagos)
- **Canais**: Geral, Conteúdo, Imagem, Vitórias, Perguntas
- **Gamification**: Observadora → Participante → Influenciadora → Embaixadora
- **Reward**: Embaixadora ganha 1 call/trimestre com Gisa

## 3. AGENTES IA (4 agentes inteligentes)

### 3.1 Agente "Diagnóstico de Posicionamento"
- 15 questões estruturadas via chat → análise IA → relatório PDF com score + gaps + recomendações
- **Valor**: Substitui 1h de call diagnóstica. Escala ilimitada.

### 3.2 Agente "Gerador de Conteúdo de Autoridade"
- Recebe briefing → 3 variações de post (LinkedIn + Instagram) com headline, corpo, hashtags, CTA
- Adapta formato por plataforma (carrossel IG, artigo LinkedIn, stories)
- **Valor**: Aluna gera conteúdo profissional sem copywriter.

### 3.3 Agente "Auditoria de Marca Pessoal"
- Recebe URLs (LinkedIn, Instagram, site) → analisa consistência, mensagem, frequência, engajamento
- Gera relatório "Auditoria de Marca" com score 0-100 e plano de ação
- **Venda avulsa**: R$297 por auditoria

### 3.4 Agente "Atendimento 24/7"
- FAQ bot + qualificação de leads + agendamento direto no calendário
- Canais: Web Chat, WhatsApp, Instagram DMs
- Handoff para humano quando necessário
- **Valor**: Speed-to-lead 24/7. Gisa não perde leads.

### Modelo de negócio dos agentes:
- **IP360**: Inclui agentes 3.1 e 3.2 como diferencial premium
- **Clube**: Agente 3.2 (10 posts/mês inclusos)
- **Avulso**: Agente 3.3 a R$297/auditoria
- **Operacional**: Agente 3.4 (reduz custo, não vendido)
- **Snapshot B2B**: Versão simplificada dos agentes 3.1 e 3.4
