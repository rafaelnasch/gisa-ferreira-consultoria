# 5. Análise de Concorrentes

## 5.1 Mapa geral do mercado

O mercado está dividido em 4 tipos de concorrentes:

### 1. Consultoras de imagem tradicionais

| | |
|---|---|
| **Força** | Estética bem trabalhada, método visual claro, fácil entendimento |
| **Fraqueza** | Superficialidade estratégica, não conectam com negócio, foco em aparência |
| **Gap que você ocupa** | Profundidade + estratégia |

### 2. Especialistas em marca pessoal (branding)

| | |
|---|---|
| **Força** | Discurso estratégico, clareza conceitual, posicionamento bem estruturado |
| **Fraqueza** | Pouca aplicação prática, não trabalham imagem e presença real, ficam no campo intelectual |
| **Gap que você ocupa** | Execução + tangibilidade |

### 3. Mentores de posicionamento genérico

| | |
|---|---|
| **Força** | Comunicação forte, promessa de resultado, conteúdo frequente |
| **Fraqueza** | Fórmulas prontas, falta de personalização, posicionamento superficial |
| **Gap que você ocupa** | Personalização + profundidade |

### 4. Especialistas em redes sociais / LinkedIn

| | |
|---|---|
| **Força** | Resultado visível (engajamento, seguidores), estratégias práticas |
| **Fraqueza** | Confundem posicionamento com conteúdo, não trabalham identidade real, foco em algoritmo |
| **Gap que você ocupa** | Essência + coerência + estratégia |

## 5.2 Diagnóstico estratégico do mercado

O mercado está **fragmentado**. Cada profissional resolve um pedaço (imagem, branding, conteúdo, posicionamento), mas quase ninguém integra tudo.

### Posição única possível

> A profissional que integra imagem + marca + comportamento + estratégia para gerar autoridade percebida real.

Em linguagem de guerra de posicionamento (Al Ries & Trout):

- Você não compete com ninguém diretamente
- Você cria uma **nova categoria mental**: "posicionamento pessoal estratégico com base em percepção"

## 5.3 Diferencial competitivo (lapidado)

| O mercado faz | Você faz |
|---|---|
| Estética | Estratégia |
| Branding | Aplicação real |
| Conteúdo | Coerência |
| Posicionamento | Percepção |

**Frase de diferenciação:**
> "Eu não trabalho partes isoladas. Eu trabalho como o mercado percebe você como um todo."

## 5.4 Refinamento da mensagem com base nos gaps

**Versão refinada pós-análise:**
> Eu ajudo mulheres especialistas a serem percebidas com o valor que realmente têm, alinhando imagem, marca pessoal e posicionamento de forma estratégica e integrada — porque não adianta parecer melhor em uma área e incoerente nas outras.

**Versão mais incisiva:**
> O problema não é falta de conteúdo, nem de imagem. O problema é desalinhamento. E é isso que impede você de ser percebida como autoridade.
